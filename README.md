# Aparajita K Singh — Developer Portfolio

Modern React + Tailwind portfolio scaffold.

Sections: Home, About, Projects, Skills, Certifications, Contact.

Setup

```bash
# macOS, zsh
cd /Users/galaxy_grid/Acads/Projects/portfolio
npm install
npm run dev
```

This project uses Vite + React + Tailwind CSS. After installing, open http://localhost:5173.

Notes
- Dark theme toggle in the navbar.
- Responsive layout and simple animations using Tailwind transitions.
